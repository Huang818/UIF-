# 用 订阅链接 导入

一般购买了代理服务，都会提供 `订阅链接`，目前 UIF 支持 `V2rayN` 、`Clash` 格式。

## 导入

到 [我的订阅](http://127.0.0.1/out/subscribe)，点击 `添加订阅` 按钮，把订阅链接填入 `订阅地址`，点击右上角 `添加` ，UIF 会自动识别 `V2rayN` 、`Clash` 格式。

## 更新

更新时需要读取目标网站的内容，因此，更新的方法有两种 `使用代理`，`不使用代理`。

假如你的订阅网站已被屏蔽，那么不使用代理更新订阅，就不能正常获取订阅内容。此时，你必须使用代理。

### 使用代理更新

使用代理的前提是UIF中已存在可用的代理服务器，此时，你只需打开`启用`开关，那么所有的订阅请求都会经过此代理服务器。

### 不代理更新

把`出站`中的代理服务器全部禁用，此时订阅更新就会走`直连`。
